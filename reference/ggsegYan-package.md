# ggsegYan: Yan 2023 Homotopic Atlas for the 'ggseg' Ecosystem

Yan 2023 homotopic brain parcellation for the 'ggseg' ecosystem.
Provides 'ggseg_atlas' objects with 2D polygon geometry and 3D vertex
indices. Includes 7-network and 17-network variants at resolutions from
100 to 1000 parcels.
