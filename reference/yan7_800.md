# Yan 7-Network 800-Parcel Atlas

Brain atlas for the Yan et al. (2023) homotopic cortical parcellation
with 800 parcels aligned to 7 resting-state networks (Yeo et al., 2011).
Contains both 2D polygon geometry for
[`ggseg::geom_brain()`](https://ggsegverse.github.io/ggseg/reference/ggbrain.html)
and 3D vertex indices for
[`ggseg3d::ggseg3d()`](https://ggsegverse.github.io/ggseg3d/reference/ggseg3d.html).

## Usage

``` r
yan7_800()
```

## Value

A
[ggseg.formats::ggseg_atlas](https://ggsegverse.github.io/ggseg.formats/reference/ggseg_atlas.html)
object (cortical).

## References

Yan X, et al. (2023). "Homotopic local-global parcellation of the human
cerebral cortex from resting-state functional connectivity."
*NeuroImage*, 273:120010.
[doi:10.1016/j.neuroimage.2023.120010](https://doi.org/10.1016/j.neuroimage.2023.120010)
